| Feature          | Description                                            |
| ---------------- | ------------------------------------------------------ |
| Filter Buttons   | Shows images category-wise using `.show()` & `.hide()` |
| Dynamic Gallery  | You can add images anytime + filtering still works     |
| Bootstrap Layout | Responsive gallery UI                                  |
| Animation        | Smooth filter transition                               |


| Functionality      | jQuery Feature Used                |
| ------------------ | ---------------------------------- |
| Filter images      | `.click()` + `.show()` + `.hide()` |
| Add images         | `.append()` DOM manipulation       |
| Get data attribute | `.data("filter")`                  |
| Get input value    | `.val()`                           |
